MIDI Controller

2018 / Passion Project / Arnhem

As a designer I'm fascinated by all the gear and devices you can use to control or influence sound on the computer (keyboards, launchpads, etc). In this 3-day project I set out to create my own device. I build it on Arduino and it uses the MIDI protocol to communicate with the computer.

The housing is made out of solid beechwood and uses an exchangeable plastic sheet as I/O shield.
