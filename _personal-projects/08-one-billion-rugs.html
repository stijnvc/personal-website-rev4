One Billion Rugs, 2020

A computer script sets the exact boundaries and parameters for a a fixed set of one billion rug designs. Although almost all of the one billion rugs are unique, they still share the same design DNA predetermined in 382 lines of computer code.

Research

The code for One Billion Rugs generates valid weaving patterns. Part of my research for writing the code was building and using a back strap loom to weave the computer generated patterns. The back strap loom is a rudimentary loom made of just sticks and string, and has been used since ancient times.
